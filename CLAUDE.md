# Инструкции для Claude Code

## Git и GitHub
- Для пуша всегда используй стандартные git-команды через bash: `git add`, `git commit`, `git push`
- НЕ используй GitHub MCP для пуша файлов — только для чтения Issues и PR если нужно
- Аутентификация настроена через GitHub CLI (gh), токен подхватывается автоматически
- Не пытайся передавать токены или credentials вручную
